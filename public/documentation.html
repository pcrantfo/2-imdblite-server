IMDBLite

This web application will allow users to access information about different movies, directors and genres. They will be able to sign up, and create a list of their favorite movies.

Endpoints

Business Logic

URL

HTTP Method

Request body data format

Response body data format
Get a list of movies /movies GET None A JSON object holding data about all movies
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, containing a title, description, genre, director, imgURL and whether it is featured. Example:

{
title: 'Lord of the Rings: Fellowship of the Ring',
description: 'A meek Hobbit from the Shire and eight companions set out on a journey to destroy the powerful One Ring and save Middle-earth from the Dark Lord Sauron.',
genre: ['action', 'adventure', 'drama'],
director: 'Peter Jackson',
imgURL: 'https://www.imdb.com/title/tt0120737/mediaviewer/rm3592958976?ref_=tt_ov_i',
featured: 'yes'
}
Return data about a genre by title /genres/[name] GET None A text message indicating the genre's description
Return data about a director /directors/[name] GET None A JSON obkect holding data about a single director, containing an ID, a name, a bio, a birth year and a death year (if applicable). Example:

{
id: 1,
name: 'Peter Jackson',
bio: 'Blah blah blah',
birthYear: '10/31/1961',
deathYear: 'N/A'
}
Register a new user /users POST A JSON object holding data about the user to add, structured like:

{
"name": "Example Name",
"username": "example",
"email": "example@gmail.com",
password: "password",
"birthDate": "1/1/1994"
}
 A JSON object holding data about the user that was added, including an ID and an empty list of favorite movies (to be updated later):

{
name: 'Example Name',
username: 'example',
email: 'example@gmail.com',
password: 'password',
birthDate: '1/1/1994',
id: 1,
favorites: []
}
Allow users to update their user info. /users/[name]/[username] PUT None A text message indicating the username has been updated.
Add a movie to a user's list of favorites. /users/[name]//[movie]/favorites PUT None A text message indicating the movie was added to the users favorites.
Remove a movie from a user's list of favorites. /users/[name]/[movie]/favorites DELETE None A text message indicating the movie was removed.
Deregister a user by email. /users/[name]/[email] DELETE None A text message indicating the user email has been removed.
